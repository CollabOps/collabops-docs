# Getting Started with CollabOps

Welcome to CollabOps! This guide will help you get up and running quickly with our team collaboration platform.

### Option 1: Cloud (Recommended)

The fastest way to get started is with our hosted solution:

1. **Sign up** at [collabops.ai](https://collabops.ai)
2. **Create your workspace** by entering your team name
3. **Invite team members** via email
4. **Start collaborating** immediately!

For organizations that need full control over their data:

1. See our [Self-Hosting Guide](/docs/self-hosting) for installation options
2. Choose between Docker, Kubernetes, or Helm chart deployment
3. Configure your environment variables and domains
4. Set up authentication and user management

## ğŸ”§ Initial Setup

### 1. Create Your First Project

After signing in:

```bash
1. Click "New Project" in your dashboard
2. Choose a project template or start from scratch
3. Set up your project details:
   - Project name
   - Description
   - Team members
   - Visibility settings
```

### 2. Configure Team Settings

```bash
1. Go to Team Settings
2. Set up your team preferences:
   - Time zone
   - Notification preferences
   - Integration settings
   - Access permissions
```

### 3. Invite Team Members

```bash
1. Navigate to Team > Members
2. Click "Invite Members"
3. Enter email addresses or share the invite link
4. Set appropriate roles and permissions
```
