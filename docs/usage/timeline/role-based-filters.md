# 역할별 보기 필터

각 팀원의 역할과 관심사에 맞춰 타임라인을 필터링하여 관련 정보만 집중해서 볼 수 있습니다.

## 👥 역할별 맞춤 뷰

### 개발자 뷰
- 본인이 담당한 이슈 및 PR 중심
- 코드 리뷰 요청 및 피드백
- CI/CD 파이프라인 상태
- 테스트 결과 및 배포 현황

### PM/기획자 뷰  
- 전체 프로젝트 진행 상황
- 마일스톤 및 스프린트 진행률
- 팀별 생산성 지표
- 리스크 및 지연 요소

### QA 엔지니어 뷰
- 테스트 대상 기능 목록
- 버그 리포트 및 재현 단계
- 테스트 환경 배포 현황
- 품질 메트릭 및 커버리지
