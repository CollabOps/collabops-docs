# 웹/이메일/웹훅 알림

중요한 프로젝트 업데이트를 놓치지 않도록 다양한 채널을 통해 실시간 알림을 받을 수 있습니다.

## 🔔 알림 채널

### 웹 알림
- 브라우저 푸시 알림
- 인앱 알림 센터
- 실시간 뱃지 및 카운터

### 이메일 알림
- 즉시 알림 vs 일괄 요약
- HTML 템플릿 커스터마이징
- 개인별 알림 빈도 설정

### 웹훅 통합
```json
{
  "event": "pull_request.merged",
  "data": {
    "pr_id": 123,
    "title": "사용자 인증 개선",
    "author": "john.doe",
    "merged_by": "jane.smith",
    "timestamp": "2024-01-15T10:30:00Z"
  }
}
```

## ⚙️ 알림 설정

### 이벤트별 알림 규칙
- 이슈 상태 변경
- PR 생성/리뷰/머지
- 배포 성공/실패
- 보안 취약점 발견
- 성능 이슈 감지

### 개인화 설정
- 업무 시간 설정
- 우선순위 기반 필터링
- 프로젝트별 구독 관리
